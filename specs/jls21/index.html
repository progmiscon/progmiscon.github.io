Java Language Specification
Java SE 21 Edition

The Java Language Specification, Java SE 21 Edition, was the official, authoritative specification of the Java programming language. It was published in August 2023 and followed the Java SE 20 Edition of the JLS.

It specifies the Java programming language as implemented in Java SE 21.

This Java SE 21 Edition goes beyond the Java SE 20 Edition of the JLS by specifying the following features:

  • Record patterns
  • Pattern matching for switch
  • Virtual Threads
  • Generational ZGC
  • Sequenced connections API
  • Key Encapsulation Mechanism API

This language change was introduced in Java SE 21 in 2023.

Read the Authoritative Specification

https://docs.oracle.com/javase/specs/jls/se21/html/index.html
